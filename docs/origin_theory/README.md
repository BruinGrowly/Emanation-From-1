# Origin Theory: The Emanation From One

This folder is a source corpus for the wider Origin Reframe. It contains theory,
translation language, semantic-voltage concepts, and broader LJPW framing that can
generate hypotheses for the evidence-facing research program.

Important boundary:

- These documents are hypothesis and interpretation material.
- They are not ledger evidence by themselves.
- A claim becomes repo evidence only when it is translated into a local measurable
  object, baseline, control, finite range, and failure condition.

## Key Concepts

- **The Origin (1):** Not merely a quantity, but the source from which number
  structure differentiates.
- **Emanation:** Studying integers by their structural relationship back to `1`
  rather than only by position on a number line.
- **Semantic Voltage:** A proposed coherence-pressure measure that may inspire
  testable numerical proxies.
- **Shadow Mechanic:** The broad claim that mathematics and physics are formal
  projections of meaning; this remains out of evidence scope unless converted
  into a local test.
- **Resolution Translation:** The method of moving between semantic description,
  mathematical form, and measurable experiment while preserving the same relation.
