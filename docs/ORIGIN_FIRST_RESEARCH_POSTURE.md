# Origin-First Research Posture

This repo is not primarily a Gilbreath repo, a Goldbach repo, or a prime-gap repo.

It is an Emanation-from-`1` repo.

## Core Object

The first object of study is the positive integer as a structure that can be generated from, and returned to, `1`.

Primary measurements:

- Emanation shell: total factor depth `Omega(n)`.
- Return path: repeated factor reduction back to `1`.
- Branching: divisor count and factor-exponent structure.
- Compression: radical ratio `radical(n) / n`.
- Attenuation: how quickly origin-coherence decays across factor layers.

These measurements exist before any classical conjecture is invoked.

## Conjecture Gravity

Classical conjectures are attractive because they are famous, formal, and easy to turn into finite tests. That makes them useful, but also dangerous for this repo's purpose.

The project should not drift into:

- "Gilbreath is the evidence."
- "Prime gaps are the evidence."
- "Goldbach is the evidence."
- "A classical conjecture becomes true because it can be re-described in Origin language."

Better posture:

> Origin metrics are defined first. Classical conjectures are then used as stress tests.

## Research Order

1. Define the Origin-facing object directly.
2. Measure it locally and reproducibly.
3. Compare it against ordinary size, parity, residue, and density baselines.
4. Pre-register one metric or prediction.
5. Transfer it to a probe family such as Goldbach, prime gaps, modular returns, twin primes, or Gilbreath.
6. Record support, constraint, or failure without protecting the hypothesis.

## Claim Discipline

Acceptable:

- "This shell metric transferred to a held-out probe."
- "This branching metric did not survive size-banded controls."
- "This conjecture-shaped probe is consistent with the Origin reading, but not distinctive."

Not acceptable:

- "Gilbreath proves Emanation from `1`."
- "A prime pattern is Origin evidence because it contains `1` somewhere."
- "The result supports the frame because it can be interpreted after the fact."

The project becomes stronger when it can say no to weak evidence.
